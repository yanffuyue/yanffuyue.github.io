hexo博客的搭建

windows系统下hexo博客的搭建

工具的安装（Node.js)

1、官网下载LTS版本（long time support 长期 支持版本）
nodejs.org

2、cmd终端验证是否安装成功

node -v
npm -v(node.js自带的管理包的工具)

3、npm镜像源切换

npm install -g cnpm –registry=https://registry.npm.taobao.org

4、cnpm 安装 hexo

cnpm install -g hexo-cl

在当前目录下创建博客文件夹

md blog
cd blog

hexo 初始化

hexo init

hexo 的使用

hexo s(start的缩写)
可对博客内容进行一个预览

hexo n(new的缩写) “helloword”
创建一篇名为helloword的博客
